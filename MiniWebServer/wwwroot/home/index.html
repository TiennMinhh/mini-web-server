About

Mini Web Server, tạo một máy chủ web đơn giản, nhanh, nhẹ, hỗ trợ nhiều tính năng:

  • Cho phép xây dựng các ứng dụng phục vụ người dùng với .NET dựa trên mô hình MVC
  • Session
  • HTTPS
  • Multi-host, cho phép phục vụ nội dung cùng lúc cho nhiều domain khác nhau
  • Hỗ trợ mapping các hàm .NET vào các request, giúp viết Web API một cách nhanh chóng

Tính năng quan trọng nhất là sẽ có người giải thích 101 câu hỏi vì sao cho các bạn muốn đọc hiểu mã nguồn
